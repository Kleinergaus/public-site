# Respectful and Sustainable Apps Website

Small static website presenting respectful, sustainable, privacy-conscious app projects.

## Pages

- `index.html`: Main landing page and developer mission
- `TaskReminder/task-reminder.html`: Task Reminder app information
- `MathMerge/math-merge.html`: MathMerge project page
- `TaskReminder/privacy-policy.html`: Privacy policy (currently for Task Reminder)
- `impressum.html`: Legal notice (Impressum)
- `datenschutzerklaerung.html`: Website privacy notice

## Suggested Repository Name

`respectful-sustainable-apps` or `sustainable-apps-studio`

## Open Locally

Open `index.html` in your browser.

## Vision

Build useful apps that avoid pay-to-win mechanics and keep data transfer minimal when possible.
