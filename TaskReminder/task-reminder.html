Task Reminder App

A lightweight to-do and reminder app with local data storage.

Features

  • Create, edit, and delete tasks quickly.
  • Set reminders to stay on schedule.
  • Export tasks manually as JSON.
  • Work offline with local-first storage.

Data Handling

Task Reminder stores tasks and settings directly on your device. No server sync is required for normal usage.
